# 隐私政策

**最后更新日期：2026年3月12日**

## 数据收集

**bimple 不收集任何用户数据。**

本扩展的所有功能均在用户浏览器本地运行，不涉及：
- 个人身份信息收集
- 浏览历史记录上传
- 用户行为追踪
- 与第三方服务器通信

## 数据存储

本扩展使用浏览器提供的 `storage` API 仅用于保存用户偏好设置（如各项功能的开关状态）。这些数据：
- 完全存储在用户的本地浏览器中
- 可通过浏览器账号同步（如果用户启用了浏览器同步功能）
- 可随时通过清除浏览器数据删除

## 权限说明

本扩展请求的权限仅用于实现核心功能：

| 权限 | 用途 |
|------|------|
| `storage` | 保存用户设置偏好 |
| `https://*.bilibili.com/*` | 在 Bilibili 页面注入样式和脚本 |
