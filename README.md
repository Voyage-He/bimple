# bimple

> 🧹 一个让 Bilibili 回归纯粹的浏览器扩展

bimple 是一个开源的浏览器扩展，通过精简 Bilibili 页面元素，帮助你专注于视频内容本身，减少无关信息的干扰。

![Version](https://img.shields.io/badge/version-0.2.0-blue)

---

## ✨ 功能特性

bimple 提供精细化的内容控制，所有功能均可通过设置面板自由开关：

### 首页优化
- **隐藏推荐内容** - 移除首页的视频推荐流，还你清爽的界面
- **隐藏搜索热点** - 去除搜索框下方的热搜词
- **清除搜索占位符** - 清空搜索框默认提示文字

### 视频页优化
- **隐藏顶部 Header** - 移除视频页顶部的导航栏
- **隐藏侧边推荐** - 移除右侧的相关视频推荐（保留分P列表）
- **隐藏评论区** - 移除视频下方的评论区域
- **隐藏播放完成推荐** - 移除视频播放结束后的推荐画面

## 📦 安装方法

### 从源码安装（开发者模式）

1. 下载或克隆本仓库
   ```bash
   git clone https://github.com/Voyage-He/bimple.git
   ```

2. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

3. 开启「开发者模式」

4. 点击「加载已解压的扩展程序」，选择本项目文件夹

5. 完成！访问 Bilibili 即可看到效果

---

## 🎛️ 使用方法

1. 点击浏览器工具栏上的 bimple 图标
2. 在弹出的设置面板中，按需开启/关闭各项功能
3. 刷新 Bilibili 页面，设置即刻生效

所有偏好设置会自动保存并同步到你的浏览器账号。

---

## 🗑️ 卸载

如需卸载本扩展：
1. 打开浏览器扩展管理页面（Chrome: `chrome://extensions/`）
2. 找到 bimple
3. 点击「移除」

---

## 🔒 隐私政策

bimple **不收集任何用户数据**。所有设置仅保存在本地浏览器中，详细说明请查看 [PRIVACY.md](PRIVACY.md)。
